Creating HatMacros


If you are having trouble choosing a hat, these codes are for you. let's start You Need a Cordinate Selector. Download by clicking Here


*$~Z::
Send, n
Sleep, 180
MouseGetPos, MX, MY
Click, Your Hat Clicking Positions
Sleep, 20
MouseMove, %MX%, %MY%
Send, n
Return

*$~Z::

When you press Z, it will add a Spike. If you want to change it, you can put any key you want instead of Z.


Send, N

"Send" is a code that is always used to send something. "Send, N" is opening Shop for you, if you changed other key then Change Shop open button instead KEY "Send, KEY"


Sleep, 180

"Sleep" Sleep waits in milliseconds. if you set Sleep it will wait for 1000 then it will wait for 1 second if you set it for 500 then it will wait for 0.5 seconds.


MouseGetPos, MX, MY

Mousegetpos of course takes the coordinates where it is on the mouse.


Click, Your Hat Clicking Positions

"Click" clicks according to your position. If Click, you make 0, 0 then it goes towards 0, 0. I am giving an example in the Shop you get from the coordinate, you will place the position of the Berserker button on this "Your Hat Clicking Positions"


MouseMove, %MX%, %MY%

If "Mousemove, 0, 0" moves towards 0, 0. Just like Click, 0, 0, but the difference of click is that after clicking on that coordinate, mousemove only goes there.


Return

"Return" is used for after the code is run
