Introduction Sploop.ahk

Sploop.ahk is made easy to use as a powerful api. Api specially tuned for our popular game Sploop.io both improves your gaming experience.


Let's start the API using these codes.


This code is used to make Autohotkey even more powerful. now let's start by putting this code in the first line



#SingleInstance Force
#NoEnv
#IfWinActive, Sploop.io - Fast paced combat game
#MaxThreadsPerHotkey 2
#MaxHotkeysPerInterval 99000000
#HotkeyInterval 99000000
#KeyHistory 0
Process, Priority, , A
SetBatchLines, -1
SetKeyDelay, -1, -1, -1
SetMouseDelay, -1
SetDefaultMouseSpeed, 0
SetWinDelay, -1
SetControlDelay, -1
weapon=1

; We will always put the codes here, so do not put the codes above!!!

*$~1::weapon=1
*$~2::weapon=2

*$~Enter::
suspend
pause,,1
Return

After installing this, please, if there is an error in the codes or codes that cannot be found, please visit on our Discord server. And tell us.
