Creating a Macro for Sploop.io


If you want to create a macro for sploop.io, these codes are right here



*$~R:: 
Send, R{space}%weapon%
Return

*$~R::

When you press R, it will add a Spike. If you want to change it, you can put any key you want instead of R.


Send, R{space}%weapon%

"Send" is a code that is always used to send something. If you type "Send, Hello world" then it will write to the R key or if there is a key that you put then when you press it it will write Hello world "Send, R" Of course it will automatically press R and you will hold the Spike in your hand "Send, R{space}" If you put the spike automatically but spam, then you will have the spike. so you can handle this by using "Send, R{Space}%weapon%".


Return

"Return" is used for after the code is run


If you want to know more about "Send", you can find out from Official Autohotkey site by Clicking Here.

Run loop code when keys are held down


For example, let's take a look at Tabiki for those who say, "I want it to keep Spike" when you hold down the R Key.


*$~R:: 
While GetKeyState("R", "P") {
Send, R{space}%weapon%
Sleep, 50
}
Return

If you want to do it with another key of this While GetkeyState("KEY", "P") of course, you can make any key instead of KEY.


Sleep "Sleep" Sleep waits in milliseconds. if you set Sleep it will wait for 1000 then it will wait for 1 second if you set it for 500 then it will wait for 0.5 seconds
